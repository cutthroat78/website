ArisAnywhere

A family of FreeBSD and Linux based operating systems for work and
personal use.

A POWERFUL NEW IDEA BY FIVNEX

Arisblu

Dekstop operating systems built with FreeBSD and Ubuntu. Using custom software, both will be usable (and soon, interchangable). Comes with Lumina, TDE, MATE, and XFCE as the desktop environments.

Arisred

Server operating systems built with FreeBSD and Debian. Using custom software, they will be usable (and soon, interchangable). Comes with any desktop supported by Arisblu, as well as without. Will use Fish as the default shell.
